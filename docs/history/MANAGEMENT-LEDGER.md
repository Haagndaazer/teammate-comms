# Management ledger — moved

The canonical management ledger is now global, the single source of truth for every manager and
implementer across all projects:

> **`~/.claude/team-rules/MANAGEMENT-LEDGER.md`**

Read and cite rules from there. New lessons are proposed/traded in the `#manager-lessons`
teammate-comms channel and ratified into that global file. This per-repo copy was retired to avoid
constraint drift (ledger rule 21 — a stale local snapshot is the hazard it warns about). The
manager operating manual sits above the ledger at `~/.claude/team-rules/rules-managers.md`.
